# xyplug-on-holiday
Execute events on holidays or prevent them from running on holidays. Holiday data is received via public API https://www.openholidaysapi.org/en/.

## Requirements
- uvx
- Python 3.9+

## Data Collection
This plugin does not collect or store any user data.
